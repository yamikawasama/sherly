# 📚 ส่วนที่ 1: คู่มือการใช้งานระบบหลังบ้าน (Admin)

## 🔐 วิธีเข้าใช้งานหลังบ้าน
1. เปิดเว็บไซต์ แล้วกดไอคอน **⚙️ แอดมิน** ที่แถบเมนูด้านซ้าย
2. ใส่ **ชื่อผู้ใช้** และ **รหัสผ่าน** ที่ตั้งไว้ (เริ่มต้นคือ `PlengloveKB` รหัสดูได้ในโค้ด)
3. กด **🔑 เข้าสู่ระบบ**
> **Tip:** สามารถเปลี่ยนรหัสได้ในหลังบ้าน หมวด **🔒 รหัส**

## 🏪 จัดการร้าน
เข้าหลังบ้าน → หมวด **🏪 ร้าน** จะมีฟังก์ชันต่อไปนี้:
- **เปิด/ปิดร้าน:** ติ๊กถูก ✅ = ร้านเปิด (แสดง 🟢), เอาติ๊กออก = ร้านปิด (แสดง 🔴)
- **ข้อความวิ่ง (Marquee):** พิมพ์ข้อความโปรโมชั่นให้วิ่งด้านบนเว็บ
- **รูปภาพต่างๆ:** แบนเนอร์, มาสคอต, หน้าโหลดเข้าเว็บ, บอทแชท, แบนเนอร์หน้าสั่งซื้อ
  > **สำคัญ:** ทุกรูปมีตัวเลื่อนปรับขนาดได้! รองรับภาพพื้นหลังใส (PNG/WebP)
- **ข้อมูลบัญชีธนาคาร:** แก้ไขชื่อบัญชี เลขบัญชี ข้อความโน๊ต (ต้องใส่รหัสแอดมินยืนยันก่อนกดบันทึก)

## 🔘 จัดการปุ่มบริการ
เข้าหลังบ้าน → หมวด **🔘 ปุ่ม**
- แก้ไขชื่อ, ไอคอน, เป้าหมายของแต่ละปุ่ม
- อัปโหลดรูปภาพแทน emoji ได้
- กด 🟢/🔴 เปิด/ปิดปุ่ม หรือ 🗑️ ลบปุ่ม

## 📦 จัดการสินค้ากระดุม
เข้าหลังบ้าน → หมวด **📦 สินค้า**
- **ชื่อ**: เช่น "66 กระดุม"
- **Echoes / โบนัส**: จำนวน Echo + โบนัสที่ได้
- **ราคาพรีไว / ราคาปกติ**: ราคาขาย
- **ต้นทุน**: ราคาทุน (ใช้คำนวณกำไร)
- **เปิดรับปกติ**: ติ๊กถูก = เปิดให้ซื้อราคาปกติ

## 🎁 จัดการสกิน (หมวดแนะนำ)
เข้าหลังบ้าน → หมวด **🎁 สกิน**
- **ชื่อแพ็ค**: เช่น "แพ็คสกินทองประดับทอง"
- **Echo ต้องการ**: จำนวน Echo ที่ต้องใช้ 
- **ตั้งราคาเอง (Custom Price)**
  - **เปิดรับปกติ:** ติ๊กถูกเพื่อเปิดปุ่มสั่งซื้อราคาปกติ พร้อมช่องตั้งราคาเอง 
  - **พรีไวตั้งราคาเอง:** ติ๊กถูกเพื่อตั้งราคาพรีไวเองนอกเหนือจากการคำนวณอัตโนมัติ

## 📋 จัดการคิว
เข้าหลังบ้าน → หมวด **📋 คิว**
- ดูรายการออเดอร์ทั้งหมด + สถานะ
- กดเปลี่ยนสถานะ: รอ → กำลังทำ → เสร็จ
- ดูรายละเอียด: สลิป, รายการสินค้า, ราคา

## 🎮 จัดการเช่าไอดี
เข้าหลังบ้าน → หมวด **🎮 เช่า**
- เพิ่ม/แก้ไข/ลบไอดีปล่อยเช่า / ตั้งราคาต่อชั่วโมง
- อัปโหลดรูปไอดี และเปิด/ปิดการเช่า

## ❓ จัดการ FAQ / โปรโมชั่น
- **FAQ:** เพิ่ม/แก้ไข/ลบคำถามที่พบบ่อย (ใส่คำตอบสั้น + เต็ม)
- **โปรโมชั่น:** เพิ่ม/แก้ไขโปรโมชั่น, อัปโหลดรูปภาพได้หลายรูป

## 🤖 จัดการบอทแชท
เข้าหลังบ้าน → หมวด **🤖 Bot**
- เพิ่มคำถาม-คำตอบใหม่ / ตั้ง Keywords (คำค้นหา) หลายคำได้

## 🔒 วิธีเปิด/ปิดสาธารณะ (ทำให้เว็บเข้าถึงไม่ได้ชั่วคราว)
- **ปิดเว็บไซต์ชั่วคราว:** ไปที่ GitHub > Settings > Pages > ตรง Branch เลือกเป็น **None** > Save
- **เปิดเว็บไซต์กลับมา:** ไปที่ GitHub > Settings > Pages > ตรง Branch เลือก **main** > โฟลเดอร์ `/ (root)` > Save
- **ซ่อนโค้ด + ปิดเว็บ (Private):** ไปที่ GitHub > Settings > เลื่อนลงสุดหา Danger Zone > Change visibility เป็น **Private**

# 🛠️ ส่วนที่ 2: คู่มือสอนแก้โค้ด (สำหรับมือใหม่!)

## 📁 โครงสร้างไฟล์
- **`index.html`** ← โครงสร้างหลักของเว็บ (กล่อง, เมนู) - แก้เวลาอยากเพิ่มเมนู
- **`css/style.css`** ← สีสัน, ความสวยงาม, บีบขยาย, ฟอนต์ - แก้เวลาอยากเปลี่ยนสี 
- **`css/loading.css`** ← ดีไซน์หน้าโหลด (กระต่าย)
- **`js/store.js`** ← ข้อมูลสินค้าตั้งต้น + ระบบเก็บข้อมูล - แก้เวลาอยากเพิ่มสินค้า
- **`js/app.js`** ← ลอจิกทั้งหมด (แสดงหน้าเว็บ, คำนวณ, แอดมิน) - แก้การทำงาน/เพิ่มปุ่ม

## 🎯 ตัวอย่างการแก้ไขโค้ดแบบรวดเร็ว

### 1️⃣ เปลี่ยนสีหลักของเว็บ
เปิด `css/style.css` ค้นหา `:root` ด้านบนสุด
```css
:root {
  --primary: #F2A7B3;        /* แก้ตรงนี้เป็นสีใหม่ เช่น #B088F9 */
  --primary-dark: #E8889A;   /* สีชมพูเข้ม */
  --bg-main: #FFF5F7;        /* สีพื้นหลัง */
}
```

### 2️⃣ เพิ่มแพ็คกระดุมใหม่ถาวร
เปิด `js/store.js` ค้นหา `DEFAULT_PRODUCTS` หาที่ว่างสร้างบรรทัดใหม่
```javascript
  {id:5, name:'1529 กระดุม', echoes:1390, bonus:139, totalEcho:1529, privatePrice:550, normalPrice:500, normalEnabled:false, cost:430, image:'', emoji:'💎'},
```
*(อย่าลืมเปลี่ยนเลย `id` ให้เรียงกันและไม่ซ้ำ!)*

### 3️⃣ เปลี่ยนชื่อร้านบนแถบหน้าแรก
เปิด `js/app.js` → กด `Ctrl + F` พิมพ์หา `𝐒𝐡𝐞𝐫ƴ𝐥`
```javascript
<div class="shop-name">୧ ‧:₊˚꒰ 𝐒𝐡𝐞𝐫ƴ𝐥 𝐏𝐚ñ𝐭ƴ ꒱ ⋅ ☆ ˖°</div>
```

### 4️⃣ เพิ่มลิงก์ใหม่ด้านล่างหน้าแรก
เปิด `js/app.js` → หา `contact-row`
```javascript
// เพิ่มลิงก์ IG เข้าไปต่อท้ายแบบนี้:
<div class="contact-row">
  <a href="..." class="contact-card">📘 <span>Facebook</span></a>
  <a href="..." class="contact-card">💳 <span>เช็คเครดิต</span></a>
  <a href="https://instagram.com/ชื่อIG" target="_blank" class="contact-card">📸 <span>Instagram</span></a>
</div>
```

### 5️⃣ ซ่อน/ลบ บอทแชท
เปิด `css/style.css` เลื่อนลงไปข้างล่างสุดและพิมพ์เพิ่ม
```css
/* ซ่อนแชทบอทถาวร */
.chatbot-container { display: none !important; }
```

## 🔍 วิธีหาจุดที่ต้องแก้ไข (ทริคสำหรับ VS Code)
1. กด `Ctrl + Shift + F`
2. พิมพ์คำที่เห็นในเว็บลงไป (เช่น "สั่งสินค้า" หรือ "กำลังโหลด")
3. VS Code จะโชว์ขึ้นมาหน้าต่างซ้ายเลยว่าโค้ดนั้นอยู่ไฟล์ไหน บรรทัดที่เท่าไหร่!

## 🐛 วิธี Debug หาจุดพิมพ์ผิด
หากแก้โค้ดแล้วหน้าเว็บไม่โหลด:
1. เปิดเว็บไซต์ พิมพ์กด `F12` หรือคลิกขวา > Inspect
2. ไปที่ช่อง **Console** ด้านบน
3. หากมีตัวหนังสือ **สีแดง** แสดงว่าเราพิมพ์โค้ดผิด (เช่น ลืมใส่ลูกน้ำ `,` หรือ ปิดวงเล็บไม่ครบ `}`) มันจะบอกบรรทัดที่เกิดปัญหาเพื่อให้เราไปแก้

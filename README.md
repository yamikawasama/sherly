# 📚 ส่วนที่ 1: คู่มือการใช้งานระบบหลังบ้าน (Admin)

## 🔐 วิธีเข้าใช้งานหลังบ้าน
1. เปิดเว็บไซต์ แล้วกดไอคอน **⚙️ แอดมิน** ที่แถบเมนูด้านซ้าย
2. ใส่ **ชื่อผู้ใช้** และ **รหัสผ่าน** ที่ตั้งไว้ (เริ่มต้นคือ `PlengloveKB` รหัสดูได้ในโค้ด)
3. กด **🔑 เข้าสู่ระบบ**

## 🏪 จัดการร้าน
เข้าหลังบ้าน → หมวด **🏪 ร้าน** จะมีฟังก์ชันต่อไปนี้:
- **เปิด/ปิดร้าน:** ติ๊กถูก ✅ = ร้านเปิด (แสดง 🟢), เอาติ๊กออก = ร้านปิด (แสดง 🔴)
- **ข้อความวิ่ง (Marquee):** พิมพ์ข้อความโปรโมชั่นให้วิ่งด้านบนหน้าแรกได้
- **รูปภาพต่างๆ:** สามารถอัปโหลดและปรับขนาด "แบนเนอร์", "มาสคอต", "ภาพตอนโหลดเข้าเว็บ", "รูปบอทแชท" และ "แบนเนอร์หน้าสั่งซื้อ/หน้าของขวัญ" ได้ทันที (ระบบจะบีบอัดรูปและรองรับพื้นหลังใส)
- **ข้อมูลบัญชีธนาคาร:** แก้ไขชื่อ, เลขบัญชี, บันทึกย่อได้ (ต้องกรอกรหัสผ่านเพื่อบันทึกส่วนนี้)

## 🔘 จัดการปุ่มบริการ & ฟังก์ชันอื่นๆ
- **หมวด 🔘 ปุ่ม:** แก้ไขชื่อ, ไอคอน, ลิงก์ปลายทาง หรือใส่รูปแทน emoji ได้ เปิด/ปิดการใช้งานปุ่มได้อิสระ
- **หมวด 📦 สินค้า:** จัดการแพ็คกระดุมแบบปกติและพรีไว (ราคา, จำนวน Echo, โบนัส, ต้นทุน) ควบคุมการเปิด/ปิดหน้าร้าน
- **หมวด 🎁 สกิน:** เพิ่มสกิน, ระบบจะคำนวณราคาให้อัตโนมัติ หรือ **ตั้งราคาเอง** ได้ทั้งแบบพรีไวและปกติ
- **หมวด 📋 คิว:** จัดการออเดอร์, เปลี่ยนสถานะ และดูสลิปโอนเงิน
- **หมวด 🎮 เช่า:** เพิ่มไอดีเช่า, ตั้งราคา, ใส่รูปไอดี ปิด/เปิดรับคิว
- **หมวด ❓ FAQ & 🎉 โปรฯ:** อัปเดตคำถามที่พบบ่อยและประกาศข่าวสาร (ใส่ภาพประกบได้)
- **หมวด 🤖 Bot:** เพิ่มคีย์เวิร์ดแชทบอทให้ตอบคำถามอัตโนมัติ

## 🔒 วิธีเปิด/ปิดเว็บสาธารณะ (ออฟไลน์/ออนไลน์)
- **ปิดเว็บไซต์ชั่วคราว:** ไปที่ GitHub > Settings > Pages > ตรง Branch เลือกเป็น **None** > Save
- **เปิดเว็บไซต์กลับมา:** ไปที่ GitHub > Settings > Pages > ตรง Branch เลือก **main** > โฟลเดอร์ `/ (root)` > Save
- **ซ่อนโค้ด + ปิดเว็บ (Private):** ไปที่ GitHub > Settings > เลื่อนลงสุดหา Danger Zone > Change visibility เป็น **Private**

# 🛠️ ส่วนที่ 2: คู่มือสอนแก้โค้ด (สำหรับมือใหม่!)

## 📁 โครงสร้างไฟล์
* `index.html` ← โครงสร้างหลักของเว็บ (กล่อง, เมนู)
* `css/style.css` ← สีสัน, ความสวยงาม, บีบขยาย, ฟอนต์
* `css/loading.css` ← โค้ดของหน้ากระต่ายตอนโหลด
* `js/store.js` ← ฐานข้อมูลตั้งต้นของสินค้า และหน้าต่างฟังก์ชันเก็บข้อมูล
* `js/app.js` ← โค้ดการกดปุ่ม, เปลี่ยนหน้า และหลังบ้านทั้งหมด

## 📝 ตัวอย่างการแก้ไขโค้ดแบบรวดเร็ว

### 1. การเปลี่ยนสีหลักของเว็บ
เปิด `css/style.css` ค้นหา `:root` ด้านบนสุด
```css
:root {
  --primary: #F2A7B3;      /* เปลี่ยนตรงนี้เป็น hex โค้ดสีใหม่ */
  --primary-dark: #E8889A;
  --bg-main: #FFF5F7;      /* สีพื้นหลัง */
}
```

### 2. การเพิ่ม/แก้ไขแพ็คกระดุมพื้นฐาน
เปิด `js/store.js` ค้นหา `DEFAULT_PRODUCTS` หาที่ว่างสร้างบรรทัดใหม่
```javascript
  {id:5, name:'1529 กระดุม', echoes:1390, bonus:139, totalEcho:1529, privatePrice:550, normalPrice:500, normalEnabled:false, cost:430, image:'', emoji:'💎'},
```
*ข้อควรระวัง: อย่าลืมเปลี่ยนเลข `id` ไม่ให้ซ้ำกัน!*

### 3. เปลี่ยนชื่อร้านบนแถบหน้าแรก
เปิด `js/app.js` → กด `Ctrl + F` พิมพ์หา `𝐒𝐡𝐞𝐫ƴ𝐥`
```javascript
<div class="shop-name">୧ ‧:₊˚꒰ 𝐒𝐡𝐞𝐫ƴ𝐥 𝐏𝐚ñ𝐭ƴ ꒱ ⋅ ☆ ˖°</div>
```

### 4. ซ่อน/ลบ น้องกระต่าย หรือแชทบอท
เปิด `css/style.css` เลื่อนลงไปข้างล่างสุดและพิมพ์เพิ่ม
```css
/* ซ่อนแชทบอทถาวร */
.chatbot-container { display: none !important; }
```

## 🔍 ทริคการใช้ VS Code เบื้องต้น
- **ค้นหาข้อความรวดเร็ว:** กด `Ctrl + Shift + F` แล้วพิมพ์คำที่เห็นในเว็บ จะรู้เลยว่าอยู่ไฟล์ไหนและบรรทัดที่เท่าไร
- **สลับหน้าต่างดูเว็บ:** กดดับเบิ้ลคลิก `index.html` บนโฟลเดอร์ในคอมเพื่อดูสิ่งที่แก้ได้ตลอดเวลาโดยไม่ต้องใช้อินเตอร์เน็ต

# 🚀 วิธีอัปเดตเว็บขึ้นออนไลน์หลังผ่านการแก้ไขโค้ด

ทุกครั้งที่แก้รหัสหรือเพิ่มฟังก์ชัน ให้พิมพ์ในโปรแกรม **Terminal (หรือ PowerShell)** ดังนี้:

```powershell
cd e:\sherly
git add -A
git commit -m "อัปเดตเว็บ"
git push
```
*(รอประมาณ 1 - 2 นาที GitHub จะประมวลผลโค้ดขึ้นระบบ และเว็บก็จะเปลี่ยนแปลงตามที่เราแก้ไว้เลยค่ะ!)*

> [!IMPORTANT]
> **ระบบความจำ (LocalStorage)**: 
> ตัวเว็บใช้ระบบพื้นฐานหน้าเว็บบราวเซอร์ในการจำข้อมูล แปลว่าถ้าคุณลูกค้า หรือผู้ใช้เคลียร์ข้อมูลประวัติเข้าชม หรือเล่นบนโทรศัพท์เครื่องใหม่ **ข้อมูลร้านค้าที่คุณตั้งใหม่/ออเดอร์ที่ถูกบันทึก จะไม่ตามไป** 
> *แต่ถ้าแก้ในตัวโค้ด เช่น `store.js` หรือ `style.css` แล้วเซฟ push ขึ้น GitHub สิ่งนี้จะอยู่ถาวร!*
